# 🌊 Rising Waters - Flood Prediction System

ML-powered flood prediction web application built with Flask.

## About
Floods are among the most devastating natural disasters. 
This system uses Machine Learning to predict flood risks 
early — giving authorities time to act and save lives.

## Tech Stack
- Python, Flask
- Scikit-learn, XGBoost
- HTML, CSS, JavaScript

## ML Models Used
- Decision Tree ✅ (Best - 100% Accuracy)
- Random Forest
- KNN
- XGBoost

## Dataset
- 115 records, 11 features
- Features: Temp, Humidity, Cloud Cover, Rainfall patterns

## How to Run
pip install -r requirements.txt
python train_model.py
python app.py
